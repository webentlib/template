SERVER
======

0. В settings_local.py в CSRF_TRUSTED_ORIGINS (и возможно `ALLOWED_HOSTS`) добавить IP сервера
1. В корне сервака создать папку www: cd / | mkdir www
2. Удостовериться что стоит git
3. Внутри www спулить проект
4. Удостовериться что стоит глобальный python 
5. Установить глобальному питону invoke: pip install invoke
6. Чтобы не вводить пароль гита по куче раз: git config --global credential.helper store https://stackoverflow.com/a/11428767/4117781
7. Установить docker, docker-compose
8. invoke re из корня проекта
9. перейти в браузер, удостовериться что всё ок
